# 更新日志

## [1.2.0] - 2026-06-05

### 新增

- **周日历**：顶部周一到周日日历条，支持 `<` `>` 切换周，点击日期筛选当日待办
  - 红点标记：该日期有待办
  - 橙色点标记：该日期有逾期未完成的待办
  - 选中日期后添加待办，自动设置截止时间为当天 23:59
- **拖拽分配日期**：拖拽待办到日历上的日期格子，自动设置截止时间
  - 循环待办不允许拖拽到日历
  - 仅在指针进入日历区域时触发，列表内拖拽仍为排序
- **多主题切换**：内置毛玻璃、深色、浅色三套主题
  - CSS 变量驱动，一键切换即时生效
  - 主题选择持久化存储，跨启动保留
- **内嵌设置弹窗**：设置页面从独立窗口改为模态弹窗
  - Switch 滑动开关替换 Checkbox
  - 自定义下拉选择替换原生 `<select>`
  - 透明度滑块优化，轨道/滑块清晰可见
- **关于弹窗**：帮助页面改为内嵌弹窗，含 B 站链接、QQ 群、微信（点击复制）

### 改进

- 主题色系统重构为 CSS 变量，三套主题共用一套变量名
- 周日历配色全线改为 CSS 变量，修复 Tailwind v4 下部分颜色失效问题
- 时钟区移至周日历上方
- 模态弹窗使用 Portal 渲染到 `document.body`，不受主面板透明度影响

### 修复

- 新增待办截止时间从 00:00:00 修正为 23:59:59，避免当日显示"逾期"
- Windows NSIS 安装包加入 WebView2 引导安装
- 循环待办图标 `pointer-events-none`，修复拖拽被图标拦截的问题

### 移除

- 独立设置窗口（`settings.html`、`SettingsForm`、`openSettingsWindow`）
- 独立帮助窗口（`help.html`、`openHelpWindow`）
- 相关未使用的 i18n 文案（`helpText`、`helpOpenFailed`）

## [1.1.0] - 2026-05-29

### 新增

- **循环待办**：支持每天/每周/每月重复提醒，可自定义具体时间和目标日
  - 每天：选择具体几点提醒
  - 每周：选择星期几 + 几点提醒
  - 每月：选择几号 + 几点提醒
  - 到达设定时间后自动进入下一轮，无需手动操作
  - 循环待办不可标记完成，只能删除
- **拖拽排序**：长按待办条目拖动即可调整顺序，5px 移动阈值避免误触
- **标题栏版本号**：左上角动态显示当前版本号

## [1.0.0] - 2026-05-23

### 功能

- 本地待办：SQLite 存储，支持新增/编辑/删除/完成
- 置顶、重要程度颜色标记
- 截止时间：快速设置（今天/明天/本周末）和自定义日期时间
- 截止提醒：到期前 15 分钟系统通知（Rust 后端 + 前端双保险轮询）
- 时钟与农历：可选显示（默认折叠）
- 无边框透明面板，毛玻璃效果，透明度可调
- 窗口置顶、关闭隐藏到系统托盘
- 全局快捷键 `Ctrl+Shift+L` / `⌘+Shift+L` 唤出/隐藏
- 独立设置页：透明度、时钟折叠、语言（中/英/跟随系统）、开机启动
- 跨平台：Windows（NSIS + 便携版）、macOS（.app / .dmg）
